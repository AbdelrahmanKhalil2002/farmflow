## الملخص التنفيذي

**فارم فلو** منصة رقمية مصممة خصيصاً لسوق المواشي المصري. تجمع بين أصحاب المزارع (البائعين) والعملاء (المشترين) من خلال سوق إلكتروني شفاف وموثوق وسهل الاستخدام لتداول المواشي ومنتجات الألبان ومستلزمات المزارع.

تخدم المنصة ثلاثة أنواع من المستخدمين:
- **التجار (البائعون)** — أصحاب المزارع الذين يديرون قطعانهم ويعرضون الحيوانات للبيع ويتابعون ماليتهم
- **المشترون** — العملاء الذين يتصفحون المزارع ويكتشفون المواشي ويضعون الطلبات
- **المشرفون (الإدارة)** — مديرو المنصة الذين يحافظون على الجودة ويوافقون على الإعلانات

فارم فلو متاح على:
- **الويب** (المتصفح — للبائعين والمشترين)
- **تطبيق الجوال** (iOS و Android — Flutter)
- **تطبيق سطح المكتب** (Windows/macOS — يعمل **أوفلاين وأونلاين**، مصمم لأصحاب المزارع الذين لديهم حاسوب)

## وصف المشروع

فارم فلو منصة متكاملة لسوق المواشي المصري تُرقمن تجارة المواشي التي كانت تتم بشكل غير رسمي. يستطيع أصحاب المزارع إدارة كل جانب من جوانب أعمالهم — من تتبع صحة كل حيوان ووزنه إلى وضع الميزانيات وتصدير التقارير المالية. ويحصل المشترون على منصة اكتشاف موثوقة تضم إعلانات موثقة وشهادات صحية وتواصل مباشر مع البائعين.

المنصة مصممة بمبادئ مصرية أولاً:
- واجهة عربية كاملة من اليمين إلى اليسار
- المحافظات المصرية وعملة الجنيه المصري
- التحقق التلقائي من الرقم القومي (يقرأ تاريخ الميلاد والعمر والمحافظة تلقائياً)
- دعم وسائل الدفع المحلية (الدفع عند الاستلام، InstaPay)

## المشكلة التي نحلها

تواجه تجارة المواشي في مصر تحديات عديدة:

| المشكلة | كيف تحلها فارم فلو |
|---------|-------------------|
| لا يوجد سوق مركزي موحد | منصة واحدة تضم جميع المزارع والحيوانات المتاحة |
| غياب الشفافية حول صحة الحيوانات | كل إعلان يعرض الحالة الصحية وسجلات التطعيم والشهادات الصحية |
| صعوبة تتبع مصاريف ودخل المزرعة | إدارة مالية متكاملة مع فئات المصاريف والميزانيات وتصدير PDF/CSV |
| لا توجد أدوات لإدارة القطيع | تتبع كامل للقطيع: سجل الأوزان، التطعيمات، السجلات الطبية، حالة الحمل |
| معاملات غير رسمية وغير موثقة | عملية موافقة إدارية، ملفات تعريف بائعين موثقة، وتدفق طلبات منظم |
| لا يوجد قناة للألبان أو المستلزمات | أقسام منفصلة لمنتجات الألبان ومستلزمات المزارع إلى جانب المواشي |

## مميزات التاجر (البائع)

### 1. لوحة التحكم

مركز قيادة التاجر. في لمحة واحدة يرى التاجر:

| الميزة | الوصف |
|--------|-------|
| بطاقات المؤشرات | إجمالي الدخل، إجمالي المصاريف، صافي الربح، وإجمالي عدد الحيوانات |
| مخطط الدخل مقابل المصاريف | مخطط خطي يُظهر الاتجاهات المالية خلال آخر 30 يوماً |
| الطلبات الأخيرة | آخر 5 طلبات من المشترين للتصرف السريع |
| تذكيرات التطعيم | الحيوانات التي يستحق تطعيمها خلال الـ 14 يوماً القادمة |
| تذكيرات الوزن | الحيوانات التي لم تُوزن خلال آخر 7 أيام |
| أزرار الإجراءات السريعة | اختصارات بنقرة واحدة: إضافة إعلان، إضافة حيوان، فتح الكشوف المالية |

### 2. إدارة القطيع

إدارة متكاملة لدورة حياة كل حيوان في المزرعة.

#### 2.1 نظرة عامة على القطيع
- عرض شبكي لجميع الحيوانات، قابل للتصفية حسب النوع (أبقار، خراف، جمال، ماعز، خيول، أرانب) أو الحالة (نشط / مباع / الكل)
- شريط ملخص: إجمالي العدد، متوسط العمر، متوسط الوزن، التوزيع حسب النوع

#### 2.2 ملف الحيوان
لكل حيوان ملف رقمي كامل يحتوي على أربع تبويبات:

| التبويب | المحتوى |
|---------|--------|
| النمو | مخطط سجل الوزن، الوزن المستهدف مع شريط التقدم، سجل إدخال الأوزان |
| التطعيمات | جميع سجلات التطعيم مع ترميز لوني لحالة الاستحقاق/التأخر |
| السجل الطبي | سجلات زيارات البيطري: التشخيص، العلاج، الأدوية، التكلفة، موعد المتابعة |
| المعلومات | السلالة، رقم الوسم، الجنس، تاريخ الميلاد، اللون، حالة الحمل |

#### 2.3 تتبع الحمل والولادة
- تحديد حالة الحمل: غير حامل / حامل / وضعت مؤخراً
- تسجيل تاريخ الوضع المتوقع
- تسجيل تفاصيل الولادة ومعلومات المواليد

#### 2.4 الشهادة الصحية
- لكل حيوان حالة صحية رسمية تظهر للمشترين
- تُضاف تكاليف العلاج تلقائياً إلى سجلات مصاريف المزرعة

---

### 3. إعلانات المواشي للبيع

يستطيع التاجر عرض حيوانات للبيع مع إبقائها في سجل القطيع.

| الميزة | الوصف |
|--------|-------|
| إنشاء إعلان | نوع الحيوان، السلالة، الوزن، السعر، خيارات التوصيل، حتى 5 صور |
| تتبع الحالة | قيد المراجعة ← موافق عليه ← مباع |
| وضع العيد | تحديد الإعلان كمتاح لموسم العيد |
| خدمة الذبح | تقديم خدمة الذبح مباشرة من خلال الإعلان |
| تعديل / حذف | تحديث أو إزالة أي إعلان مع تأكيد |

> **مهم:** فقط الحيوانات التي يحددها التاجر صراحةً للبيع تظهر للمشترين. باقي الحيوانات تبقى سجلات قطيع خاصة.
